🚀 5 DAKİKADA YAYINLAYIN!

1️⃣  SUPABASE KURULUMU (2 dk)
   → https://supabase.com → Hesap oluştur
   → New Project → Proje oluştur
   → Settings → API → URL ve Key'i kopyala
   → SQL Editor → SUPABASE_SETUP.sql'i yapıştır ve çalıştır
   
2️⃣  VERCEL'E DEPLOY (2 dk)
   → https://vercel.com → Hesap oluştur
   → New Project → ZIP yükle veya GitHub'dan import et
   → Environment Variables ekle:
      • NEXT_PUBLIC_SUPABASE_URL
      • NEXT_PUBLIC_SUPABASE_ANON_KEY
   → Deploy!

3️⃣  İÇERİK EKLE (1 dk)
   → Sitenize gidin: https://site-adiniz.vercel.app/admin
   → Proje, yetenek, deneyim ekleyin
   → Ana sayfada görün!
